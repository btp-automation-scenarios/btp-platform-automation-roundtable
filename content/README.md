# Landing Page for SAP BTP Platform Automation Roundtable

This page serves as a central landing page for the sessions of the SAP BTP Platform Automation Roundtable focusing on the BTP CLI and Terraform. It provides links to session recordings, slides, and related resources.


The [GitHub Pages documentation website](https://github.com/btp-automation-scenarios/btp-platform-automation-roundtable) is built with :blue_heart: using [Docusaurus 3](https://docusaurus.io/), a modern static website generator.
